== Description ==

This is a Webservice to test if your webservers return state 200. The service will send a notification email in worst case.

== Installation ==

This section describes how to install the plugin and get it working.

   1. Upload files to the /wp-content/plugins/server-down/ directory.
   2. Activate the plugin through the ‘Plugins’ menu in WordPress.
   3. Setup your server-down account. 
   
   <strong>Done.</strong>.
